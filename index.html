MascotaFeliz — Sistema de Gestión Veterinaria

MascotaFeliz es un sistema web diseñado a partir del relevamiento y análisis de necesidades de una clínica veterinaria, con el objetivo de mejorar la organización de turnos y el seguimiento clínico de las mascotas mediante un registro digital centralizado.

Durante el análisis se detectaron problemáticas como la ausencia de historial médico digital, falta de asignación de profesionales y gestión manual de turnos, lo que generaba demoras y pérdida de información.

Rol

Analista de Sistemas (Proyecto académico aplicado). Participación en la definición de requerimientos funcionales, diseño de artefactos de análisis y desarrollo de un prototipo navegable para validar los flujos principales del sistema.

Tecnologías y herramientas

HTML · CSS · JavaScript · UML · Análisis de Requisitos
